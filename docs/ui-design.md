# UI 设计规范

## 1. 视觉方向

整体参考 Apple 官网 / Apple Health：简洁、高级、自然、轻盈。

关键词：浅色背景、留白充足、圆角卡片、柔和阴影、健康绿色点缀、干净字体层级、数据卡片化、图表柔和。

## 2. 色彩规范

| 用途 | 建议色值 | 说明 |
|---|---|---|
| 页面背景 | #F7F8F6 | 温和浅灰白 |
| 主色 | #34C759 | 健康绿色，接近 Apple Health 气质 |
| 主文字 | #1D1D1F | 高级黑 |
| 次级文字 | #6E6E73 | 柔和灰 |
| 卡片背景 | #FFFFFF | 白色卡片 |
| 边框 | #E5E5EA | 轻边框 |
| 警示色 | #FF9F0A | 温和橙色 |
| 错误色 | #FF3B30 | 系统红 |

## 3. 组件风格

- 卡片圆角建议 18-24px。
- 页面大块区域保持宽松间距。
- 不使用厚重边框和强烈阴影。
- 表格优先简洁，避免密集后台系统风格。
- 首页以健康指标卡片、环形进度和柔和趋势图为主。
- 表单控件保持大尺寸、圆角、清晰标签。
